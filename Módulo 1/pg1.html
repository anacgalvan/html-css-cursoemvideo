Ordenadas

Não Ordenadas

Para listas que não precisam de numeração, usamos as listas não ordenadas, com a tag UL. Exemplo:

  • Aulas de Programação
  • Aulas de Idiomas
  • Academia
  • Ler 3 capítulos

Lista de Definições

Usamos esse tipo de lista para colocar termos e definições, e para ela as tags são DL, DT e DD. Essa lista é importante especialmente para mecanismos de busca.

ABNT

CSS
Linguagem de marcação para o design de um site.
