Hierarquia de Títulos

Cada título tem uma ordem hierárquica, que vai de H1 a H6. É importante utilizar o título certo para a ordem do título, por motivos de semântica (ou seja, forma fica com o CSS, significado com o HTML).

Principais formatações:

Negrito / Destaque

Nessa frase, temos um termo em negrito com a tag B (tag não semântica). E nessa frase, temos um termo em destaque, usando a tag STRONG.

Itálico / Ênfase

Nessa frase, temos um termo em itálico usando a tag I (tag não semântica). E nessa, um termo em ênfase, usando a tag EM.

Texto Marcado

Podemos criar também um texto marcado criando a tag MARK.

Texto Grande e Pequeno

Estamo criando um texto grande e um texto pequeno, com as tags BIG (obsoleta) e SMALL.

Texto Deletado

Podemos marcar um texto como excluído, ou seja, ele deve ser lido mas não consideradom usando a tag DEL.

Texto Inserido

Podemos marcar um texto como inserido, o que dá ênfase e indica que ele foi adicionado posteriormente, usando a tag INS.

Existe ainda a ênfase sublinhada usando a tag U, mas ela não é semântica.

Texto Sobrescrito e Subscrito

Texto Sobrescrito é para inserir coisas como x2+3, usando a tag SUP.

Texto Subscrito é para inserir textos como H2O, usando a tag SUB.

Outras Formatações e Código-Fonte

O comando document.getElementById('test') é escrito em linguagem JavaScript. Para usar essa formatação, usamos a tag CODE.

O exemplo abaixo usa as tags PRE e CODE para inserir um código em Pithon com indentação:

        
    num = int(input('Digite um número'))
    if num % 2 == 0:
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('Fim do programa')

Citações

Como diria o pai de um amigo: o computado é um burro muito rápido. Para inserir a citação de forma semântica, usamos a tag Q.

Para inserir citações completas, ou seja, citações longas, usamos a tag BLOCKQUOTE:
