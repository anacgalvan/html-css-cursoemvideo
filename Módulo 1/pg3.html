Imagens Dinâmicas

Aqui, usamos as tags PICTURE, SOURCE e IMG para inserir imagens dinâmicas, que se ajustam ao tamanho da tela.

Áudios

Para inserir áudios com HTML no seu site, é possível usar a tag AUDIO.

Vídeos Locais

Para inserir vídeos hospedados localmente (no próprio servidor), use a tag VIDEO. Porém, vídeos hospedados dessa maneira vão consumir muito tráfego de dados.

Vídeos Externos

Também é possível inserir vídeos externos ao seu site, usando a ferramenta de incorporar dos sites de vídeo. Portanto, é interessante ter seus vídeos postados no YouTube (ou qualquer outro site, como o Vimeo ou TikTok) e usar essa ferramenta para incorporá-los, sem consumir tráfego de dados do seu site.
